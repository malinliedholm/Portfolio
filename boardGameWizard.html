Board Game Wizard

Board Game Wizard is an interactive visualization of board games, where you can find board games by filter on things like game type, playtime, game mechanics and much more. The project was a group project developed in the course DH2321 Information Visualization at KTH.

During the project it was my first-time programming in JavaScript as well as my first time using the D3.js JavaScript library, so I learned a lot about both during the project. My main contribution in the project was to implement the interactive legend for game type, structure the information about each board game in the information container that shows up when pressing a board game in the scatterplot. I also worked on filtering of number of players, playtime, recommended player age and number of ratings

If you want to check out and explore the project for yourself, you can do it here: Board Game Wizard
